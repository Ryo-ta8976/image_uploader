PNG画像をドラッグ&ドロップすると画像URLを生成してくれる。

# Features
 
フルスタックエンジニアになるためのアプリ制作第一弾。

- ファイルのドラッグ&ドロップ
- ファイルをフォルダから選択可能
- 画像アップロード
- 画像URLのコピー
 
# Requirement
 
"hoge"を動かすのに必要なライブラリなどを列挙する
 
* huga 3.5.2
* hogehuga 1.0.2
